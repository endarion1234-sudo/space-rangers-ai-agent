# Space Rangers AI Agent

Автономный ИИ-агент для управления кораблем в игре "Космические рейнджеры", созданный с использованием машинного обучения с подкреплением (Reinforcement Learning).

## 🎯 Описание проекта

Этот проект демонстрирует создание автономного агента, способного принимать решения и управлять игровым процессом без участия человека. Агент обучается с помощью алгоритма PPO (Proximal Policy Optimization) от библиотеки Stable Baselines3 и использует компьютерное зрение (YOLOv8) для анализа состояния игры.

Проект служит отличным примером интеграции RL-агента с реальной средой через системные вызовы и управление курсором мыши.

## Технологический стек

*   **Язык программирования:** Python 3.x
*   **Машинное обучение:** Stable Baselines3, PyTorch
*   **Компьютерное зрение:** Ultralytics YOLOv8, OpenCV
*   **Управление системой:** D3DShot, pygetwindow, ctypes
*   **Среда разработки:** Gymnasium

## Установка и запуск

Для запуска проекта выполните следующие шаги:

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/endarion1234-sudo/space-rangers-ai-agent.git
    cd space-rangers-ai-agent
    ```

2.  **Создайте виртуальное окружение и установите зависимости:**
    *(Рекомендуется использовать venv или conda)*
    ```bash
    python -m venv .venv
    source .venv/bin/activate  # На Windows используйте `.venv\Scripts\activate`
    pip install -r requirements.txt
    ```

3.  **Настройка:**
    *   Убедитесь, что игра запущена.
    *   Проверьте путь к модели YOLO в файле `config.yaml`.

4.  **Запуск:**
    ```bash
    python main.py
    ```

> **Примечание:** Для работы скрипта требуется обученная модель YOLOv8. Вы можете обучить её самостоятельно или использовать готовую.

##  Архитектура проекта

*   **Environment (`SpaceRangersEnv`)**: Реализует интерфейс Gymnasium, связывает все компоненты и вычисляет награду.
*   **Agent**: Обертка вокруг модели PPO, управляет загрузкой/сохранением модели.
*   **VisionSystem**: Использует YOLOv8 для обнаружения игровых объектов (кораблей, станций) на экране.
*   **Pilot**: Физически перемещает курсор мыши в окне игры.

##  Что я реализовал в этом проекте?

*   Разработал модульную архитектуру RL-агента.
*   Интегрировал несколько сложных библиотек для решения задачи управления в реальном времени.
*   Настроил систему логирования и обработки исключений.
*   Реализовал механизм сохранения и загрузки моделей для продолжения обучения.
Проект находится в активной разработке. Текущая реализация является прототипом и имеет следующие ключевые ограничения:

*   **Упрощенная среда:** Агенту подается изображение низкого разрешения (64x64), которое служит временной заглушкой. Ведется работа над переходом к семантической карте окружения для более точного восприятия.
*   **Недостатки архитектуры агента:**
    *   Отсутствует связность признаков: Без реализации механизма внимания (*Attention*) агент не может фокусироваться на ключевых объектах (например, угрозах или целях).
    *   Отсутствие памяти: В текущей модели нет слоев RNN/LSTM, из-за чего агент страдает от "амнезии", не способен анализировать последовательности действий и строить долгосрочную стратегию.
    *   Невозможность гибридного управления: Архитектура не позволяет эффективно комбинировать разные типы пространства действия (например, нельзя реальзя реализовать векторное непрерывного действия вместе с дискретным).

## 🚧 Дорожная карта развития

Для преодоления вышеуказанных ограничений было принято архитектурное решение отказаться от стандартных политик в пользу разработки собственной кастомной модели.

Планируется внедрение следующих компонентов:
1.  ***MLP-ветка (Multi-Layer Perceptron):*** Для обработки структурированных данных о ресурсах, технологиях и состоянии корабля.
2.  ***RNN/LSTM слои:*** Для добавления памяти и способности анализировать временные зависимости в игровом процессе.
3.  ***Механизм Attention:*** Для концентрации агента на наиболее важных элементах семантической карты.
4.  ***CNN:*** Для обработки семантической карты и извлечения состояния среды. 


Этот проект демонстрирует мои навыки в области Reinforcement Learning, Computer Vision и системного программирования на Python.
